13 Masternode enhancements
14 -----------------------
15 
16 When restarting the client the masternodes will be stored in the datadir in
17 *mncache.dat* file (make sure you don't have any wallet with such name or this
18 functionality will not be available for you)

21 Mining and relay policy enhancements
22 ------------------------------------
23 
24 Dash Core's block templates are now for version 3 blocks only, and any
25 mining software relying on its `getblocktemplate` must be updated in parallel
26 to use libblkmaker either version 0.4.2 or any version from 0.5.1 onward. If you
27 are solo mining, this will affect you the moment you upgrade Dash Core,
28 which must be done prior to BIP66 achieving its 951/1001 status. If you are
29 mining with the stratum mining protocol: this does not affect you. If you are
30 mining with the getblocktemplate protocol to a pool: this will affect you at the
31 pool operator's discretion, which must be no later than BIP66 achieving its
32 951/1001 status.
33 
34 
35 BIP 66: strict DER encoding for signatures
36 ------------------------------------------
37 
38 Dash Core 0.11.2 implements BIP 66, which introduces block version 3, and a
39 new consensus rule, which prohibits non-DER signatures. Such transactions have
40 been non-standard since Dash 0.8, but were technically still permitted
41 inside blocks.
42 
43 This change breaks the dependency on OpenSSL's signature parsing, and is
44 required if implementations would want to remove all of OpenSSL from the
45 consensus code.
46 
47 The same miner-voting mechanism as in BIP 34 is used: when 751 out of a
48 sequence of 1001 blocks have version number 3 or higher, the new consensus
49 rule becomes active for those blocks. When 951 out of a sequence of 1001
50 blocks have version number 3 or higher, it becomes mandatory for all blocks.
51 
52 Backward compatibility with current mining software is NOT provided, thus
53 miners should read the first paragraph of "Mining and relay policy
54 enhancements" above.
55 
56 Also compare with [upstream release notes](https://github.com/bitcoin/bitcoin/blob/0.10/doc/release-notes.md#mining-and-relay-policy-enhancements).
57 
58 More info on [BIP 66](https://github.com/bitcoin/bips/blob/master/bip-0066.mediawiki).

61 How to Upgrade
62 --------------
63 
64 If you are running an older version, shut it down. Wait until it has completely
65 shut down (which might take a few minutes for older versions), then run the
66 installer (on Windows) or just copy over /Applications/Dash-Qt (on Mac) or
67 dashd/dash-qt (on Linux).
